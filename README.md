# Steam Download Tracker

Скрипт для отслеживания скорости загрузки игр в Steam в фоновом режиме.

## Возможности

- ✅ Автоматическое определение пути установки Steam (через реестр Windows)
- ✅ Считывание скорости загрузки каждую минуту
- ✅ Отображение информации о скачиваемой игре
- ✅ Определение статуса загрузки (скачивание, пауза, завершено)
- ✅ Работа в фоновом режиме
- ✅ Логирование всех событий
